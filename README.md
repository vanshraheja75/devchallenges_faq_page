## Overview

![Thumbnail for the Simple FAQ Page coding challenge](./thumbnail.jpg)

This is my responsive solution for the Simple FAQ Page challenge. The goal was to build a clean, functional Frequently Asked Questions section that matches the design provided by DevChallenges.

### Built With

-   HTML5
-   CSS3 (Custom properties, Flexbox/Grid)
-   Responsive Design (Mobile-first workflow)

## Features

-   **Responsive Layout:** The page adapts seamlessly to mobile, tablet, and desktop screens.
-   **Interactive Elements:** Users can click questions to reveal or hide answers.
-   **Clean UI:** Matches the spacing, colors, and typography of the original design.

## How to Run Locally

To run this project on your local machine:

1.  Clone the repository:
    ```bash
    git clone https://github.com/vanshraheja75/devchallenges_faq_page.git
    ```
2.  Navigate to the project directory.
3.  Open `index.html` in your browser.
